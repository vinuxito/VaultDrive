# QUICKSTART — ABRN Drive Development

> Drop this file to get started immediately with ABRN Drive development.

---

## TL;DR — Essential Commands

```bash
cd /lamp/www/ABRN-Drive

# Frontend work (React/TypeScript)
cd vaultdrive_client
npm run build
sleep 10  # Wait for build + auto-reload
# Now test

# Backend work (Go)
cd /lamp/www/ABRN-Drive
go build
sudo systemctl restart abrndrive
# Now test

# SQL work (use sqlc ONLY!)
# 1. Edit: sql/queries/*.sql
# 2. Generate: sqlc generate
# 3. Rebuild: go build && sudo systemctl restart abrndrive

# Database connection
psql "postgres://postgres:postgres@localhost:5432/vaultdrive?sslmode=disable" -c "SELECT version();"
```

| Component | Path |

| **Database** | PostgreSQL 16, connection in `.env` |

---

## Key Rules

1. **Frontend build takes 8-10s, WAIT 10s before testing**
2. **All SQL queries MUST use sqlc** (never write raw SQL in Go)
3. **Never write `sqlc generate` output** — it gets regenerated
4. **Database credentials**: Read from `/lamp/www/ABRN-Drive/.env`
